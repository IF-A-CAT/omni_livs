# Ground Truth Calculation

This document describes the current ground truth calculation strategy for the
device project.

## Outdoor Open Scenarios

In open outdoor scenarios, the project prioritizes RTK positioning results as
the ground truth reference. The RTK result is generated by the u-blox ZED F9P
GNSS receiver.

The ZED F9P four-constellation RTK solution has the following nominal accuracy:

| Direction | Nominal Accuracy |
| --- | --- |
| Horizontal | 0.01 m + 1 ppm |
| Vertical | 0.02 m + 1 ppm |

The RTK trajectory should be used as the preferred position reference when RTK
fix quality is stable and the outdoor environment provides a clear satellite
view.
